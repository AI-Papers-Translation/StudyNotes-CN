# 参考文献

## 推荐书目

-   PRML：模式识别与机器学习，Bishop。
    -   通过概率的角度增强对机器学习的理解，覆盖面够宽泛，需要较多的数学知识
-   西瓜书：机器学习，周志华。
    -   本科教学用书，网上参考资料丰富，覆盖面够宽泛，但是许多地方都只有简单的说明。
-   小蓝书：统计学习方法，李航。
    -   初始入门，并不简单，公式较多，推导和解释不足，网上参考资料丰富。
-   模式分类：Duda
    -   经典老书，辅助理解许多机器学习和模式分类的知识，对许多概念可以追根溯源。
