常用的工具软件

-   Typora：用于编辑 Markdown 文件，可以有 所见即所得 的效果，特别是对数学公式。
-   VSCode：用于编辑各种文件，友好的界面，强大的控制，丰富的插件，可以整合文档的编写过程。
-   pandoc：用于将 Markdown 文件转换成需要的文本格式，例如：PDF、PPTX、DOCX 等等。
